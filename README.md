# 🧠 GenAI Price Optimization Assistant

## 📌 Overview
This project is a GenAI-powered system that recommends optimal product pricing based on demand, competition, and stock levels.

## 🚀 Features
- Price optimization
- Competitor-based pricing
- Demand & stock analysis
- AI-generated explanation (GenAI)

## 🧠 GenAI Usage
Used LLM to generate business explanations for pricing decisions.

## 🏗️ Tech Stack
- FastAPI
- OpenAI
- Python

## 📊 Output
- Recommended price
- Pricing strategy explanation
